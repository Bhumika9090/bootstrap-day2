Introduction to Bootstrap

  1. It is a css frame work with some js, to build responsive web design and mobile first approach
  2. It was developed by twitter
  3. It is a frame work contains re-usable components of HTML,CSS and JS

Key features

  1. Responsive web design
  2. Predefined components
  3. Customizable
  4. Cross-browser compatibility
  5. Open source
  6. Grid-system

Use cases

  1. Single page applications/canding page
  2. Responsive webdesign
  3. Dashboards
  4. E-Commerce websites
